加速客服處理速度

客服人員不在需要一步一步的操作系統，並且一層一層的點選標籤，而是會由AI直接提供標籤類別，使用本系統可以減去了中間花費的許多時間，加速整體客服人員的服務速度，也能夠縮減顧客平均的等待時間。

提升客戶滿意度
